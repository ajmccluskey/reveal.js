  • LAMBDA UNDERPANTS

Promoting FP in the Enterprise

  • Korny Sietsma (@kornys) had pretty graphs
  • More detailed account of parallel language experiment from previous Meetup
  • Java vs Scala vs Clojure. Clojure won

Optional Clojure Type Checking

  • Honours project of Ambrose Bonnaire-Sergeant (@ambrosebs)
  • Clojure type checking via a library - inspired by typed Racket
  • Can play with it now - https://github.com/clojure/core.typed
    • 
(clojure.core.typed/ann v t) ; gives var v the static type t.
(clojure.core.typed/ann-form f t) ; ensures form f is of the static type t.
(clojure.core.typed/check-ns) ; type checks the current namespace.
(clojure.core.typed/cf t) ; type checks the form t.
